# Backend дипломного проекта "Movies explorer"

### Функции сервера:

* Создание нового пользователя
* Редактирование инфо пользователя

* Добавление фильма в Избранное
* Запрос списка фильмов
* Удаление фильма по id

### Технологии:

* Git
* Webpack
* npm
* node.js
* express.js
* mongo DB
